# Правила для ИИ (AI Rules)

## 1. Локализация и Текст (i18n)

- **ЗАПРЕЩЕНО** хардкодить текст напрямую в компонентах (например, `<div>Привет</div>`).
- Весь UI-текст должен быть обёрнут в хук `useTranslation` и функцию `t()` из `react-i18next`.
- При добавлении нового текста, необходимо добавить переводы для ключей в оба словаря (`ru` и `en`) в соответствующие JSON файлы в папках `frontend/src/i18n/locales/ru/` и `frontend/src/i18n/locales/en/`.
- Группируйте ключи по namespace/модулю (например, `explain_ui.something`, `navbar.something`).
- Специфичные технические термины (PostgreSQL операции: *Seq Scan, Cost, Buffer*) не переводятся на русский, оставляйте их на английском языке.

Более подробная инструкция лежит в `frontend/src/i18n/docs.md`.
